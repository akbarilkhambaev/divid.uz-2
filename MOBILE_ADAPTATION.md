## 📐 Стандарты адаптации:

### Размеры экранов:
```css
/* Mobile First подход */
Default: 320px - 767px (мобильные)
sm: 640px
md: 768px (планшеты)
lg: 1024px (маленькие ноутбуки)
xl: 1280px (десктопы)
2xl: 1536px (большие экраны)
```

### Типографика:
```
Мобильный → Десктоп
- Заголовки H1: text-3xl → text-6xl
- Заголовки H2: text-2xl → text-4xl
- Заголовки H3: text-xl → text-3xl
- Текст: text-base → text-lg
```

### Отступы:
```
Мобильный → Десктоп
- Контейнеры: px-4 → px-8 → px-12
- Секции: py-8 → py-12 → py-16
- Элементы: gap-4 → gap-6 → gap-8
```

### Сетки:
```
Мобильный → Планшет → Десктоп
- grid-cols-1 → grid-cols-2 → grid-cols-3/4
- flex-col → md:flex-row
```

## 🎨 Рекомендации:

### 1. **Использовать Mobile First подход**
```jsx
// Сначала стили для мобильных, потом для больших экранов
<div className="text-sm md:text-base lg:text-lg">
```

### 2. **Тестировать на реальных устройствах**
- iPhone SE (375px)
- iPhone 12/13/14 (390px)
- Samsung Galaxy (360px)
- iPad (768px)
- iPad Pro (1024px)

### 3. **Оптимизация изображений**
- Использовать responsive images
- Lazy loading для изображений
- WebP формат где возможно

### 4. **Производительность**
- Минимизировать анимации на мобильных
- Использовать CSS вместо JS где возможно
- Отложенная загрузка компонентов

### 5. **Touch-friendly**
- Минимальный размер кнопок: 44x44px
- Достаточные отступы между элементами
- Hover эффекты не обязательны на мобильных
