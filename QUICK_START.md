# 🎉 Быстрый старт системы администрирования

## 🚀 Первый запуск (после установки)

### 1. Перейдите на страницу первоначальной настройки:
```
http://localhost:3000/admin/setup
```

### 2. Создайте первого супер-администратора:
- **Username**: admin
- **Email**: admin@divid.uz  
- **Password**: ваш_пароль (минимум 6 символов)

### 3. Войдите в систему:
После создания вы будете перенаправлены на `/admin/login`

---

## 📱 Быстрые ссылки

- **Вход**: `/admin/login`
- **Настройки**: `/admin/settings`
- **Главная админки**: `/admin`

---

## 🔐 Управление администраторами

### Добавить нового администратора:
1. Войдите в систему
2. Перейдите в "Настройки" (боковое меню)
3. Нажмите "Добавить администратора"
4. Заполните форму

### Роли:
- **Администратор** - доступ к управлению контентом
- **Супер-администратор** - полный доступ + управление админами
